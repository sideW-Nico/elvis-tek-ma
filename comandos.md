# Comandos Git

- `git init`: Crea un repositorio en un directorio a nivel local.

- `git add <ARCHIVO>...`: Agrega archivos a la etapa de staging.

- `git status`: Muestra los cambios realizados en la etapa de staging.

- `git commit -m "COMANDO"`: Genera un commit con un mensaje.

- `git push`: Sube a la nube todos los commits.

- `git pull`: Descarga de la nube los ultimos cambios.
